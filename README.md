# LucidPlasma

![LucidPlasma Screenshot](./screenshot.png)

## Installation

lazy.nvim:

```lua
'ADJB1212/lucidplasma.nvim',
lazy = false,
priority = 1000,
config = function()
  vim.cmd.colorscheme "lucidplasma"
end
```
